User Page: Ken Cenerelli - TechNet Articles - United States (English) - TechNet Wiki

Ken Cenerelli is a programmer writer who specializes in creating high quality technical publications from complex software architecture. He has over 14 years of experience as a software developer and engineers strong, data-driven web applications using the Microsoft .NET Framework. Ken is a Microsoft Most Valuable Professional (MVP) for his work with the .NET Platform in the developer community.
