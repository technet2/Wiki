Lex currently works as a BizTalk consultant at Axon Olympus. He works in the Information Technology for more than 25 years. He had roles like helpdesk engineer, system administrator, technical application engineer, functional designer, .NET developer and architect. He has worked with BizTalk Server since BizTalk Server 2004. His first BizTalk assignments were as a developer, but since more than 6 years he has mainly worked as a BizTalk administrator. Lex has a passion for developing and he maintains some BizTalk Administration related tools, namely BizTalk Processing Monitor and BizTalk Message Decompressor.

BizTalk Innovation Day

BizTalk Innovation Day is a one-day event focused purely on Microsoft BizTalk Server related topics, that has been conducted in several major European cities since February 2011: Nieuwegein (The Netherlands), Milan (Italy), Stavanger (Norway), London (UK) and Porto (Portugal).



Lex has been co-host and speaker of the event in Nieuwegein (The Netherlands) and a guest speaker of the event in Stavanger (Norway).



You can follow BizTalk Innovation Day through its Facebook page: https://www.facebook.com/BizTalkInnovationDay

Other Contributions

Lex is a member of the board of the Dutch BizTalk User Group and is working to further evolve this User Group. To support the members and the organisation of the User Group Lex created and maintains a SkyDrive which contains the presentations from the sessions being held at the Dutch BizTalk User Group.

As a result of his BizTalk Events activities, Lex is the initiator and co-founder of the Belgian BizTalk User Group (http://www.btug.be).
