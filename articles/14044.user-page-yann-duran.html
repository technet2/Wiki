Yann Duran is self-employed, self-taught software developer, based in Australia, who has been involved in writing software, in one form or another, for over 25 years.

Over the years, he taught himself a number of software technologies, including (most recent first):

  • LightSwitch (B1, B2, 2011 RTM, 2012 B1, 2012 RC, 2012 RTM)
  • Windows Presentation Foundation (WPF)
  • Entity Framework (EF)
  • WinForms
  • VB.NET
  • SQL Server (T-SQL)
  • MS Access (VBA)
  • Turbo Pascal

He was awarded the Microsoft Community Contributor Award in 2011 for contributions to the LightSwitch community.
